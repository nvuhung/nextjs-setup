The code is inspired from: https://gitlab.com/kachkaev/website-frontend

1. Install npm dependencies using Yarn or Npm

    ```bash
    yarn
    ```

2.  Copy `.env.example` to `.env`.

3.  Start the server in `development`

    ```bash
    yarn dev
    ```

4.  Build and run in `production`:

    ```bash
    yarn build
    yarn start
    ```
